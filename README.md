# Embedded AI System for Autonomous Vehicle

## Project Overview

This project is an Embedded AI-based Autonomous Vehicle System designed to perform real-time navigation using computer vision and artificial intelligence techniques. The system processes live camera input, detects lanes and obstacles, and makes driving decisions autonomously.

The project combines:

* Embedded Systems
* Artificial Intelligence
* Computer Vision
* Real-Time Processing
* Autonomous Navigation

---

# Features

* Real time Object detection and obstacle recognition
* Autonomous navigation logic
* Real-time camera processing
* Embedded AI implementation
* Vehicle movement control simulation
* provide safe navigation
* reduce speed while raining

---

# Technologies Used

* Python
* OpenCV
* TensorFlow / Keras
* NumPy
* Embedded AI Hardware ( Raspberry Pi)
* Computer Vision Algorithms
